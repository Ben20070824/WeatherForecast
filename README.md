实现功能：
    可通过输入城市名称，查询某城市的天气，包括当日天气，七日天气和31日天气
    还可以为自己设计一个主页（虽然有点简陋）

实现思路：
    子线程网络请求，通过Handler实现线程通信，将返回的JSON数据，发送给UI线程
    主页保存和账号注册通过SharedPreferences保存
    头像的保存涉及权限申请
    主页和我的切换通过ViewPager2和BottomNavigationView实现
    查看天气的选项通过TabLayout和ViewPager2实现
    （有个滑动冲突没有解决）

心得体会：
    bug有点难找，activity和fragment的生命周期感觉还有点问题，然后感觉还是得规范代码，可能为空的就加个验证，否则一不小心就崩了。
    最头大的是之前页面显示不出的问题，排查了好久找不到原因，主要还是两个关键点没做好。
    一是activity和fragment的生命周期没吃透，ViewPager2切换时没设置缓存，Fragment被销毁重建，数据就丢了，页面自然空白；
    二是RecyclerView没做好校验，更新数据后没调用刷新方法，就算数据请求到了，也显示不出来。
